# Contributing Guidelines (COMP2156 Group Assignment)

## Branch Naming
Create your own branch using:
STUDENTID-Name

Example:
101234567-JaneDoe

## Work Requirements
- Each member must make at least 10 commits.
- Each member must create 3 files:
  - studentid_gb.txt
  - studentid_devops.txt
  - studentid_sdlc.txt

## Rules
- Do all edits locally (do NOT edit directly in the GitHub website editor).
- Commit often with clear messages.
- Push your branch to GitHub.
- Open a Pull Request (PR) to merge into main.
