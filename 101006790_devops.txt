The COMP2156 DevOps course teaches important concepts used in modern software development. 
DevOps combines development and IT operations to improve collaboration, efficiency, and software reliability. 
In this course, students learn how to use Git and GitHub to manage code, create branches, and work on projects as a team.
Version control allows developers to track changes, avoid conflicts, and collaborate safely.
DevOps also includes Continuous Integration (CI) and Continuous Deployment (CD).
